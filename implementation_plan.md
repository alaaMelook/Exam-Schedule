# إصلاح الموظفين الاحتياطيين

## المشاكل المُلاحظة

1. **الاحتياطيون مش باينين في عرض حسب الموعد** — الإحصائية بتقول `0` احتياطيون بالرغم إن الـ feature موجودة
2. **الإضافة اليدوية بتعرض كل الموظفين** — لما بتختاري التاريخ والنطاق، المفروض يظهر بس الموظفين اللي مش معينين أساسي في نفس الوقت ده
3. **اختيار موظف عشوائي مش بيتسجل** — ممكن يكون بسبب constraint في الداتابيز (UNIQUE) أو إن الـ upsert بيفشل بدون رسالة خطأ
